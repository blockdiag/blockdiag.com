sphinxcontrib-blockdiag is sphinx extension for embedding blockdiag diagrams. You can embed block diagrams with blockdiag directive.

.. blockdiag::

    diagram admin {
      top_page -> config -> config_edit -> config_confirm -> top_page;
    }
